# Vijaya_sankar_ecommerse_order_engine

## Overview
This project is a *ecommerce backend engine* implemented using Python,Django,RestFramework .  
It simulates real-world backend systems like Amazon/Flipkart :

## Features

### Product Management
- Add products with price and stock
- Prevent duplicate product IDs
- Real-time stock updates

### Cart System
- Multi-user cart support
- Add/remove items
- Stock reservation on add

###Order Management
- Place orders from cart
- Cancel orders
- View all orders

### CLI OPERATIONS
1. Add Product
2. View Products
3. Add to Cart
4. Place Order
5. Cancel Order
6. Simulate Concurrency
7. View Orders
0. Exit
